# java-fundamentals-course

Repositorio de código fuente del curso **Fundamentos de Java para Principiantes**, elaborado como parte del trabajo complementario del curso 1ASI0729 Desarrollo de Aplicaciones Open Source — Universidad Peruana de Ciencias Aplicadas, período 202610.

**Plan del curso completo**: [course-plan.md](./course-plan.md)

---

## Configuración

No se requiere instalar ninguna herramienta. Todo el código se ejecuta en editores en línea gratuitos desde el navegador:

- [Replit](https://replit.com) — Lecciones 1, 2 y 5
- [JDoodle](https://www.jdoodle.com/execute-java-online/) — Lección 3
- [OnlineGDB](https://www.onlinegdb.com) — Lección 4

Para ejecutar el código localmente (opcional):
1. Tener instalado [Java JDK 17+](https://www.oracle.com/java/technologies/downloads/)
2. Compilar: `javac NombreArchivo.java`
3. Ejecutar: `java NombreArchivo`

## Estructura del repositorio

```
java-fundamentals-course/
|
|-- README.md                  <- Este archivo
|-- course-plan.md             <- Plan completo del curso con lecciones y enlaces
|
|-- starter-files/             <- Archivos de práctica para los estudiantes
|   |-- Lesson1_HelloWorld.java
|   |-- Lesson2_Variables.java
|   |-- Lesson3_ControlFlow.java
|   |-- Lesson4_Methods.java
|   +-- Lesson5_OOP.java
|
|-- completed-examples/        <- Soluciones completas de referencia
|   |-- Lesson1_HelloWorld_done.java
|   |-- Lesson2_Variables_done.java
|   |-- Lesson3_ControlFlow_done.java
|   |-- Lesson4_Methods_done.java
|   +-- Lesson5_OOP_done.java
|
+-- scripts/                   <- Notas de cada lección para el instructor
    +-- lesson-notes.md
```

### starter-files/
Archivos Java con el esqueleto del código. Los estudiantes los abren en el editor en línea y completan las secciones marcadas con `// TODO`. No incluyen la solución a propósito.

### completed-examples/
Versiones completamente funcionales de cada ejercicio. Sirven como referencia para el estudiante después de intentar resolverlo por su cuenta.

### scripts/
Notas de apoyo para el instructor con los puntos clave de cada lección y sugerencias de enseñanza.
